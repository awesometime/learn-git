TCP/IP协议族按照层次由上到下，层层包装。
```
应用层       http，ftp
表示层                        将数据从主机特有的格式转换为网络标准传输格式
会话层                        采用何种连接发送
传输层       TCP，UDP         确立连接与断开连接，重发
网络层       IP               它负责对数据加上IP地址和其他的数据（后面会讲到）以确定传输的目标。
数据链路层                    这个层次为待传送的数据加入一个以太网协议头，并进行CRC编码，为最后的数据传输做准备。
物理层       MAC地址          负责网络的传输，这个层次的定义包括网线的制式，网卡的定义等等.
```
