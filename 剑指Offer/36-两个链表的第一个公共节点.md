### 方法一
```python3
# 若两个链表长度相等, 那么遍历一遍后，在某个时刻，p1 == p2
# 若两个链表长度不等, 那么短的那个链表的指针pn（也就是p1或p2）必先为null，
#                     那么这时再另pn = 长链表头节点。经过一段时间后，
#                     则一定会出现p1 == p2。
# 如果没有公共节点：这种情况可以看成是公共节点为null，故不用再考虑

# 0-1-2-3-4-5-null
# a-b-4-5-null
# 如果有公共结点，那么指针一起走到末尾的部分，也就一定会重叠。
# p1： 0-1-2-3-4-5-null(此时遇到ifelse)-a-b-4-5-null
# p2:  a-b-4-5-null(此时遇到ifelse)-0-1-2-3-4-5-null     从4开始找到4为公共结点


class Solution:
    def FindFirstCommonNode(self, pHead1, pHead2):
        p1, p2 = pHead1, pHead2
        while p1 != p2:
            p1 = p1.next if p1 else pHead2
            p2 = p2.next if p2 else pHead1
        return p1
```
### 压入栈再弹出
		
依次将链表中的元素压入两个栈中，然后每次从两个栈中抛出一个元素，直到抛出的结点不同时停止
返回后面的元素都是公共的
```python3
class Solution:
    def FindFirstCommonNode(self, pHead1, pHead2):
        # write code here
        lst1 = []
        lst2 = []
        result = []
 
        if not pHead1 or not pHead2:
            return None
 
        p1 = pHead1
        p2 = pHead2
 
        while p1:
            lst1.append(p1)
            p1 = p1.next
        while p2:
            lst2.append(p2)
            p2 = p2.next
 
        while lst1 and lst2:
            node1 = lst1.pop()
            node2 = lst2.pop()
            if node1 == node2:
                result.append(node1)
         
        if result:
            node = result.pop()
            return node
			


    def FindFirstCommonNode2(self, pHead1, pHead2):
          if not pHead1 or not pHead2:
              return None

          stack1 = []
          stack2 = []

          while pHead1:
              stack1.append(pHead1)
              pHead1 = pHead1.next

          while pHead2:
              stack2.append(pHead2)
              pHead2 = pHead2.next

          first = None
          while stack1 and stack2:
              top1 = stack1.pop()
              top2 = stack2.pop()
              if top1 is top2:   # is
                  first = top1
              else:
                  break
          return first
```	
### 先让长的链表走m-n个节点
```
# 首先依次遍历两个链表，记录两个链表的长度m和n，假设m>n，就先让长的链表走m-n个节点，
# 然后两个链表同时遍历，遍历到相同节点的时候停止即可。

class Solution:
    def FindFirstCommonNode(self, pHead1, pHead2):
        if not pHead1 or not pHead2:
            return None
        
        p1,p2 = pHead1,pHead2
        len1 = len2 = 0
        while p1:
            len1 += 1
            p1 = p1.next
            
        while p2:
            len2 += 1
            p2 = p2.next
            
        if len1 > len2:
            while len1 - len2:
                pHead1 = pHead1.next
                len1 -= 1
        else:
            while len2 - len1:
                pHead2 = pHead2.next
                len2 -= 1
                
        while pHead1 and pHead2:
            if pHead1 is pHead2:     # 内存级别的相等
                return pHead1
            pHead1 = pHead1.next
            pHead2 = pHead2.next
            
        return None
```
