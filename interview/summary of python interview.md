#### 1 Python的函数参数传递

类型是属于 对象(可变、不可变)的，而不是变量
```
a = 1                a为变量，也叫引用，对地址的引用
def fun(a):          1是对象------strings, tuples, numbers是不可更改的对象
    a = 2                 |------list, dict, set           是可以修改的对象
fun(a)               当一个 引用a 传递给函数的时候, 函数自动复制一份引用, 这个函数里的引用和外边的引用没有半毛关系了
print a  # 1
```
